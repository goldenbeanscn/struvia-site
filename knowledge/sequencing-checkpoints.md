# Sequencing Checkpoints

Sequencing checkpoints define when work must stop for verification before the next exterior panel step proceeds.

## Checkpoint Rule

Do not cover, repeat, seal, or fix a detail until the active control point has been inspected.

## Before Install Checks

- Substrate condition is suitable for fastening and support.
- Panels are dry, undamaged, and properly handled.
- Cut lines and trim sequence are confirmed.
- Flashing, drainage, and accessory requirements are understood.
- Site has a cleaning plan for filings, dust, and debris.

## During Install Checks

- Interlock is seated before fixing.
- Screw line reaches proper support.
- Trim protects cut edges without reversing water path.
- Drainage exits remain open after panel seating.
- Field cuts are cleaned before moisture exposure.

## After Install Checks

- Joint line is continuous.
- Trim closure is stable and aligned.
- Drainage path is open.
- Cut edges are protected.
- Photos capture close detail and wide context.

## Correction Workflow

1. Stop repeated installation at the first failed checkpoint.
2. Identify the physical cause: gap, blockage, reverse lap, distortion, exposed cut, debris, or support issue.
3. Correct the detail before covering it.
4. Reinspect the corrected control point.
5. Record evidence.
6. Resume only after the sequence is stable.

## Prevention Workflow

- Convert repeated failures into pre-install checks.
- Add evidence requirements for vulnerable details.
- Review first-bay installation before full production.
- Use the same checkpoint language across related pages.
