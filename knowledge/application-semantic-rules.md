# Application Semantic Rules

# Purpose

This document defines the semantic boundaries for architectural application categories used across:

* assets
* projects
* applications
* SEO
* AI image naming
* homepage generation
* visual systems

The goal is to maintain consistent architectural understanding across the AI website system.

---

# Residential Categories

## Prefab Houses

### Definition

Factory-prefabricated residential buildings designed for permanent or semi-permanent living.

### Characteristics

* modular-friendly
* scalable housing systems
* fast-install workflows
* industrialized construction

### Includes

* prefab villas
* prefab residential systems
* modular prefab housing
* permanent prefab homes

### Excludes

* transportable mobile structures
* temporary movable cabins

### Semantic Keywords

* prefab
* modular housing
* exterior systems
* residential systems

---

# Agricultural Categories

## Poultry House

### Definition

Agricultural livestock or poultry structures designed for environmental efficiency.

### Standard Naming

Always use:

poultry-house

Avoid:

* poultry-building
* poultry-facility
* poultry-system

### Semantic Keywords

* poultry
* agricultural building
* insulated livestock structure

# Naming Rules

## File Naming Standard

Use:

* lowercase
* kebab-case
* semantic architectural wording

Example:

modern-prefab-house-exterior.jpg

Avoid:

* IMG_001.jpg
* final.png
* image-new.jpg
