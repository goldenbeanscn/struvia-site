# Coverage Gap System

The coverage gap system tracks what is missing or weak across the Technical Center.

## Gap Categories

### Missing Evidence

Occurs when a page has technical claims or workflow logic but lacks visual proof.

Examples:

- No real installation photos.
- No QC images.
- No project evidence.
- No handling visuals.
- Only proxy images available.

### Weak Node Systems

Occurs when node pages exist but lack engineering depth.

Examples:

- Thin node description.
- No failure chain.
- No inspection logic.
- No evidence protocol.
- No related execution page.

### Incomplete Failure Logic

Occurs when the page explains correct installation but not failure behavior.

Examples:

- No cause-condition-risk chain.
- No correction workflow.
- No prevention rule.
- No risk visualization.

### Inspection Gaps

Occurs when a page cannot guide verification.

Examples:

- Missing before install checks.
- Missing during install checks.
- Missing after install checks.
- No acceptance criteria.
- No evidence review gate.

### Weak Execution Depth

Occurs when a page remains overview-level.

Examples:

- Generic system description.
- No sequence.
- No node behavior.
- No field constraints.
- No workflow consequences.

## Gap Tracking Table

| Gap Type | Severity | Typical Fix |
|---|---|---|
| Missing evidence | High | Add annotated real or representative evidence |
| Weak node system | High | Add node logic, failure chain, inspection checks |
| Incomplete failure logic | High | Add cause-condition-risk-correction chain |
| Inspection gap | High | Add before/during/after verification workflow |
| Weak execution depth | Medium | Add sequencing and field behavior |

## Current Known Gaps

- Exact head flashing photos are still needed.
- Exact base drainage outlet photos are still needed.
- Some node pages remain thinner than the newest technical subpages.
- Project pages use representative evidence where exact project sets are not available.
- Thermal continuity still needs cut-edge and fastener bridge close-ups.

## Gap Closure Rule

A gap is closed only when the page contains the missing engineering layer and that layer is connected to evidence or inspection logic.
