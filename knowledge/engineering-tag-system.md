# Engineering Tag System

The engineering tag system gives Technical Center pages consistent retrieval labels.

Tags should describe engineering behavior, execution stage, risk, evidence type, or inspection purpose.

## Core Tags

### WATER CONTROL

Use for pages that explain rain shedding, overlap, flashing, moisture management, or drainage.

Related pages:

- Waterproof logic.
- Window flashing.
- Base drainage.
- Drainage exit node.
- Project evidence for rain-exposed buildings.

### THERMAL CONTINUITY

Use for pages that explain insulation continuity, fastener bridges, trim bridges, openings, joints, or thermal data interpretation.

Related pages:

- Thermal envelope.
- Thermal continuity joints.
- Product structure.
- Thermal performance logic.

### QC

Use for factory, production, dimensional, surface, packaging, or evidence verification content.

Related pages:

- Manufacturing & QC.
- Factory system.
- QC system.
- Project evidence for factory buildings.

### DRAINAGE

Use for lower water release, base drainage, sill outlets, drainage holes, drying paths, or blocked outlet risks.

Related pages:

- Base drainage.
- Drainage exit node.
- Waterproof logic.
- Window flashing.

### INSTALLATION STEP

Use for pages that define order of work, sequencing gates, installation rhythm, or correction workflow.

Related pages:

- Interlock sequence.
- Installation systems.
- Sequencing checkpoints.
- Site handling.

### FAILURE RISK

Use for content that explains what can fail, why it fails, and how it is prevented.

Related pages:

- Failure architecture.
- Project risk prevention.
- Inspection system.
- Technical subpages.

### INTERLOCK

Use for panel engagement, receiving edge, joint seating, screw fixing, and repeated bay rhythm.

Related pages:

- Interlock sequence.
- Interlocking joint node.
- Installation systems.

### AIRFLOW

Use carefully for designed ventilation, drying paths, cavity logic, or airflow interruption. Do not imply rainscreen behavior unless the assembly is designed for it.

Related pages:

- Waterproof logic.
- Thermal envelope.
- Risk visualization system.

### FLASHING

Use for opening head, jamb, sill, water-shedding lap, and project-specific waterproofing coordination.

Related pages:

- Window flashing.
- Waterproof logic.
- Opening trim evidence.

## Tagging Rules

- Use tags to support retrieval, not decoration.
- A page may have multiple tags.
- Tags should map to user problems and inspection questions.
- Do not use performance tags unless the page includes the relevant technical logic.
- Evidence pages should include both system tags and evidence tags.
