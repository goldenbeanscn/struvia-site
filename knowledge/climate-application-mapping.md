# Climate Application Mapping

Climate application mapping links project environments to exterior system risks and inspection priorities.

The mapping does not replace project engineering, code review, or climate-specific wall design. It organizes the Technical Center's evidence requirements by likely climate pressure.

## Climate Risk Factors

### Hot-Humid / Rain-Exposed

Common in Southeast Asia and coastal regions.

Primary concerns:

- Frequent rain exposure.
- High humidity.
- Moisture retention.
- Drainage path reliability.
- Cutting debris corrosion.
- Ventilation and drying where designed.

Inspection focus:

- Overlap direction.
- Flashing closure.
- Base drainage.
- Sill release.
- Clean lower trims.

### Hot-Dry / High Solar

Primary concerns:

- Thermal exposure.
- Surface temperature.
- Expansion movement.
- Fastener and trim stress.

Inspection focus:

- Joint continuity.
- Trim restraint.
- Thermal transition review.
- Surface finish condition.

### Cold / Mixed Climate

Primary concerns:

- Thermal bridging.
- Condensation risk at cold surfaces.
- Air leakage at joints.
- Opening transitions.

Inspection focus:

- Core continuity.
- Joint seating.
- Fastener bridge review.
- Opening and trim transitions.

### High Wind / Open Site

Primary concerns:

- Facade stability.
- Fastener restraint.
- Edge uplift.
- Corner exposure.

Inspection focus:

- Screw zone and substrate.
- Interlock seating.
- Corner trim restraint.
- Edge closure.

### Modular / Transported Project

Primary concerns:

- Handling deformation.
- Transport impact.
- Joint movement.
- Site reassembly.

Inspection focus:

- Loading protection.
- Edge condition after transport.
- Installation sequence after arrival.
- Final alignment and trim closure.

## Application Mapping By Project Type

### Southeast Asia Warehouse

Likely climate pressure: hot-humid, rain-exposed, large wall fields.

Priority checks: drainage, overlap, base termination, cleaning, wind restraint.

### Prefab Office

Likely project pressure: speed, modular coordination, openings, transport.

Priority checks: installation sequence, window flashing, trim closure, thermal continuity.

### Factory Building

Likely project pressure: large wall areas, wind exposure, repeated bays, QC control.

Priority checks: screw zone, interlock seating, panel rhythm, substrate support.

### Modular House

Likely project pressure: handling, transport, site reassembly, opening transitions.

Priority checks: edge protection, corner trims, window flashing, thermal continuity.

### Commercial Facade

Likely project pressure: visual alignment, openings, corners, public-facing finish.

Priority checks: trim continuity, corner closure, drainage at openings, project evidence.
