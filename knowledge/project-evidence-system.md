# Project Evidence System

Project evidence converts real-world project execution into engineering validation for the Technical Center.

Projects are not galleries. A project page should show why a system was selected, how it was installed, what risks were controlled, and what evidence supports the execution logic.

## Evidence Purpose

Project evidence should validate:

- System selection logic.
- Installation sequencing.
- Waterproof strategy.
- Thermal continuity review.
- Buildability benefits.
- Inspection checkpoints.
- Failure prevention.
- Execution correction workflow.

## Required Project Evidence Layers

### 1. Climate Logic

Explain the project environment and likely envelope pressures:

- Rain exposure.
- Heat and solar load.
- Humidity.
- Wind exposure.
- Site handling constraints.
- Construction speed or labor constraints.

### 2. System Selection Logic

Explain why an exterior panel system is suitable for the project type without exaggerating performance.

Selection should reference:

- Lightweight installation.
- Repeatable panel sequencing.
- Trim and edge closure.
- Thermal control where panel construction supports it.
- Waterproof detailing through overlap, flashing, and drainage.
- Export, logistics, or modular buildability when relevant.

### 3. Installation Evidence

Project evidence should include:

- Before-install condition.
- Installation sequence frame.
- Trim or interlock close-up.
- Fixing or screw-zone evidence.
- Completed wall context.

### 4. Risk Prevention

Each project should identify likely failure risks and prevention steps:

- Trapped moisture.
- Wrong overlap.
- Blocked drainage.
- Thermal bridge.
- Unsupported edge.
- Cutting corrosion.
- Handling deformation.

### 5. Inspection Checkpoints

Each project should define:

- Before install checks.
- During install checks.
- After install checks.
- Correction workflow.
- Evidence capture requirement.

## Project Page Rule

A project page should prove execution logic. If a photo does not show a control point, it should be labeled as context evidence, not detail evidence.
