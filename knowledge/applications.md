# Applications

# Applications Positioning

Struvia exterior systems are designed for modern construction scenarios requiring:

* simplified installation
* exterior efficiency
* modular compatibility
* fast-track construction
* scalable workflows
* modern architectural appearance

Applications should consistently communicate:

* buildability
* construction efficiency
* modern exterior systems
* simplified execution
* industrialized construction logic

Avoid positioning applications as:

* low-end temporary structures
* cheap construction systems
* generic wall material usage

---
