# Role-Based Navigation

Role-based navigation maps professional intent to engineering knowledge routes.

This is not audience marketing. It is a retrieval model for helping each role find the Technical Center pages that answer their technical questions.

## Architect

Primary concerns:

- Product structure.
- Thermal envelope behavior.
- Waterproof detailing.
- Opening and corner nodes.
- Project evidence.

Useful routes:

- `technical-center/product-structure/`
- `technical-center/thermal-envelope/`
- `technical-center/waterproof-logic/`
- `technical-center/waterproof/window-flashing/`
- `technical-center/trim/outside-corner/`
- `technical-center/project-evidence/commercial-facade/`

Typical queries:

- How does the panel layer behave thermally?
- What happens at window openings?
- How should trim relate to waterproofing?
- What project evidence supports the system?
