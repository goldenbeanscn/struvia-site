# Search Intent Map

## SEO Goal

Build a search-intent-driven construction system website.

The website should align page structure with real user search behavior.

Avoid creating pages without search intent relevance.

---

# Product Research Intent

## Keyword Examples

* metal siding panels
* insulated metal siding panels
* exterior wall panels
* wall cladding systems

## User Intent

Users are researching products and comparing exterior systems.

Users want to know:

* appearance
* installation method
* durability
* applications
* modern design compatibility

## Recommended Page Type

Systems Page

## Recommended Content

* Product categories
* Exterior visuals
* Structure diagrams
* Installation system
* Real project usage

---

# Homepage Search Intent

The homepage should support:

* Product understanding
* Installation understanding
* Application understanding
* Project trust
* Contact conversion

The homepage should NOT behave like:

* a product dump page
* a traditional factory catalog
* a low-quality trading company website

---

# Semantic SEO Direction

The website should build semantic relevance around:

* Buildability
* Exterior Systems
* Fast-install Systems
* Modern Construction
* Construction Efficiency
* Modular Construction
* Simplified Exterior Installation

Avoid semantic positioning such as:

* cheap materials
* bulk wholesale
* low-cost panels
* commodity supplier
