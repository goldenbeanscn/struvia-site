Trim / Outside Corner

Outside corner execution reference.

Plane-change trim logic for edge protection, corner alignment, impact exposure, overlap continuity, and two-face inspection.

Execution Scope

Corner node: first plane, return plane, trim closure, water path, impact check.

Engineering Overview

Outside corners combine visible alignment, exposed cut edges, water exposure, and impact risk. Both wall planes must be checked together before the corner trim is closed.

Failure Risks

Open Edge

Cut panel edge remains exposed at the corner.

Plane Drift

Return wall misaligns even when the first face appears straight.

Corner Impact

Corner trim or panel edge deforms during handling or service.

Assembly Sequence

01 / Establish Corner Line

Set vertical datum and check substrate on both planes.

02 / Close First Plane

Install first panel field without leaving exposed edge.

03 / Coordinate Return

Align return wall panel rhythm before final trim closure.

04 / Install Trim

Protect both cut edges and inspect water-shedding direction.

Node Behavior

  • Corner trim protects both cut edges.
  • Both planes must remain aligned.
  • Water should shed around, not into, the corner.
  • Impact risk is higher than flat wall field.

Inspection Checkpoints

Two-Face Alignment

Check both faces and corner line.

Edge Coverage

No exposed core or cut steel at corner.

Trim Restraint

Trim is fixed and does not rattle.

Water Path

No upward-facing gap at corner closure.

Common Failure Conditions

One-Face Inspection

Only the main wall is checked, return plane drifts.

Exposed Cut

Trim does not fully cover panel edge.

Reverse Closure

Trim edge catches water and routes it behind panel.
