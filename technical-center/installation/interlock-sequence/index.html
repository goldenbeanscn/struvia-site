Installation / Interlock Sequence

Interlock sequence execution reference.

Panel-to-panel engagement logic for receiving edge preparation, seating, screw-line control, joint inspection, and repeatable facade rhythm.

Execution Scope

Installation node: clean edge, seat panel, fix screw line, inspect joint, repeat bay.

Engineering Overview

Interlocking is a sequence, not a visual effect. The next panel must fully engage the receiving edge before the screw line locks the relationship into place.

Failure Risks

Partial Seating

Joint appears closed but is not fully engaged.

Premature Fixing

Screws lock a misaligned panel before correction.

Edge Damage

Deformed interlock edge prevents continuous engagement.

Assembly Sequence

01 / Clean Edge

Remove dust, filings, packaging debris, or deformation from receiving edge.

02 / Align Panel

Bring panel into position without dragging or twisting.

03 / Seat Interlock

Confirm engagement along the run before fixing.

04 / Fix & Inspect

Fasten only after seating, then inspect gap and surface distortion.

Node Behavior

  • Interlock controls alignment and edge exposure.
  • Screw zone controls restraint.
  • Substrate controls load transfer.
  • Inspection controls repeatability.

Inspection Checkpoints

Engagement

Joint is seated continuously.

Screw Zone

Fastener line hits suitable substrate.

Surface Plane

No distortion from over-fixing.

Repeat Rhythm

Panel run stays aligned across bays.

Common Failure Conditions

False Closure

Visible gap looks acceptable but interlock is not seated.

Over-Fixing

Screw pressure distorts panel face or joint.

Accumulated Error

Small panel rhythm error repeats across the facade.
