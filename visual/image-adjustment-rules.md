# Image Adjustment Rules

## Core Visual Principle

Prioritize adjusting real project images instead of generating artificial architectural visuals.

The website must feel:

- architectural
- realistic
- premium
- minimal
- buildability-focused
- system-oriented

Avoid AI-generated landing page aesthetics.

---

# Visual Identity Direction

The homepage should represent:

- exterior construction systems
- modern architectural workflows
- real-world buildability
- scalable installation systems
- architectural applications across multiple building types

The visual language should feel closer to:

- architectural firms
- facade system brands
- construction system companies

NOT:

- prefab cabin catalog
- tiny house blog
- AI-generated architecture gallery
- decorative product marketplace

---

# Image Selection Rules

## Prioritize

1. Real project photography
2. Real architectural applications
3. Real installation environments
4. Real exterior facade visibility
5. Real construction logic
6. Modern architectural composition
7. Exterior systems visible from realistic angles
8. Building scale and context

---

## Avoid

- abstract mockups
- fake CGI renders
- unrealistic luxury imagery
- texture-only graphics
- over-designed marketing visuals
- isolated material renders
- collage-heavy compositions
- unrealistic lighting
- excessive visual effects

---

# Homepage Image Logic

Homepage imagery should communicate:

- buildability
- installation efficiency

- scalable applications

The homepage must NOT feel like:

- a product catalog
- a decorative panel gallery

# Image Composition Rules

## Preferred Composition

- facade-focused
- architectural-centered
- installation-visible
- realistic environmental context
- balanced foreground/background
- moderate environmental depth
- readable building geometry

---

## Avoid

- excessive empty sky
- excessive ground area
- tiny building subjects
- cluttered construction debris
- visual distractions
- over-cropped corners
- extreme perspective distortion

---

# Image Adjustment Rules

When original images are imperfect, prefer:

- recropping
- composition adjustment
- facade enlargement
- perspective balancing
- brightness balancing
- shadow softening
- reducing visual clutter
- subtle environmental cleanup
- realistic color balancing

---

## Allowed Adjustments

- remove watermarks
- remove distracting logos
- remove accidental color blocks
- extend sky naturally
- extend ground naturally
- remove isolated visual distractions
- rebalance image proportions

---

## Avoid Excessive Editing

Do NOT:

- redesign buildings
- generate fake architecture
- heavily stylize lighting
- over-sharpen
- over-saturate
- create unrealistic shadows
- create CGI appearance

---

# Overlay Rules

## Preferred

- soft overlays
- minimal opacity
- subtle text backgrounds
- clean typography contrast

---

## Avoid

- oversized black blocks
- aggressive gradients
- dark-heavy sections
- excessive transparency layers
- boxed collage overlays

---

# Homepage Layout Rhythm

The homepage should feel:

- clean
- spacious
- architectural
- structured
- readable
- premium

Use:

- fewer but stronger images
- larger architectural focal points
- consistent visual rhythm
- clean spacing hierarchy

Avoid:

- fragmented image mosaics
- excessive small cards
- uneven image heights
- visual overcrowding

---

# Image Ratio Rules

## Application Cards

Use:

- 4:3 aspect ratio
- object-fit: cover

Purpose:

- maintain architectural readability
- keep facade proportions balanced
- improve grid rhythm consistency

---

## Hero Images

Preferred ratios:

- 4:5
- 5:4

Hero images should:

- emphasize facade scale
- maintain architectural atmosphere
- support typography placement

Avoid:

- ultra-wide hero images
- tiny building subjects
- collage hero compositions

---

## Industrial Images

Prefer:

- wider facade visibility
- centered structural composition
- reduced empty sky
- visible construction logic

---

## Commercial Images

Prefer:

- storefront visibility
- facade rhythm
- entrance clarity
- human-scale realism

---

## Residential Images

Prefer:

- balanced environment
- realistic exterior context
- facade detail visibility
- warm architectural atmosphere
